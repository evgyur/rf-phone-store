# RF Phone Store

Static prototype of a Russian mobile phone shop focused on ten popular smartphone models.

## What is inside

- Home page with featured offer blocks.
- Catalog with product cards, real phone images, prices, specs, and CTAs.
- Product detail route for each smartphone.
- Model comparison page.
- Contact and order request page.
- Required shop information pages: payment, delivery, warranty, returns, privacy policy, user agreement, and seller details.

## Run locally

Open `index.html` directly in a browser, or run a static server:

```bash
npx serve .
```

Then open the local URL printed by the server.

## Notes

The product list and prices are demo content. Before using the site commercially, verify actual stock, prices, legal seller details, and current demand data for the Russian market.
